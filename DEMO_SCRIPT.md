# Ghost Demo Recording Script

**Goal:** a 60-second high-energy video for the Mozilla Fellowship application.

## Scene 1: The Void (0:00 - 0:10)

- **Visual:** Black screen.
- **Audio:** Silence.
- **Action:**
  - Type `ghost-start` in a terminal window.
  - Cut to: **The Neural Terminal** (Landing Page) appearing with the "Boot Sequence" typing animation.
- **Voiceover:** "Identity is sovereign. Intelligence should be too."

## Scene 2: The Interface (0:10 - 0:30)

- **Visual:** Mouse hovering over the 3D StarField. The "Ghost Chip" logo pulses.
- **Action:**
  - Scroll down to the "Capabilities" Bento grid.
  - Hover over "Local Inference" to show the glow effect.
- **Voiceover:** "Meet Ghost. A local-first architecture that runs entirely on your device."

## Scene 3: The Engine (0:30 - 0:50)

- **Visual:** Split screen. Code editor (VS Code) showing `ghost.proto` on left, Terminal executing a voice command on right.
- **Action:**
  - User says: "Ghost, summarize my last email."
  - Terminal shows: `[Kernal] Intent: SUMMARIZE_EMAIL -> Approved`.
- **Voiceover:** "Zero telemetry. Kernel-level permission gating. You hold the keys."

## Scene 4: The Call (0:50 - 1:00)

- **Visual:** Return to the "Neural Terminal" hero section.
- **Action:** Click the "Initialize Protocol" button.
- **Voiceover:** "Ghost. The Hollow Prism. Available now on GitHub."
